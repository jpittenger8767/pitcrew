## Robot Overview {#robot}

**TBD** is our competition robot for the REBUILT season. Designed for speed, precision, and reliability, TBD combines innovative mechanisms with battle-tested engineering.

### Drivetrain
- **Type:** Swerve drive for omnidirectional movement
- **Motors:** 4x MAXSwerve Modules 
- **Wheels:** 3 in Custom 3D Printed
- **Speed:** 5.1 m/s top speed, optimized for quick cycles
- **Features:** Wheel-position based odometry, NAVX for heading correction

### Scoring Mechanism
- **Shooter** Dual-flywheel shooter with indexer
- **Actuator:** 2 NEO Motors with RPM smart control
- **Precision:** ±100 RPM
- **Speed:** Max RPM in 4 seconds

### Intake System
- **Design:** Slap-down intake
- **Collection:** Active roller system 
- **Capacity:** ~50 fuel
- **Cycle Time:** 20 seconds full hopper

### Control System
- **Controller:** Roborio
- **Vision:** 2x Arducam with Orange Pi 5 coprocessor 
- **Communication:** NI Driver Station

### Code Highlights
```java
public class PathfindToVision {

    public static Command create(
            DriveSubsystem drive,
            Pose2d tagPose) {
        // Offset so we don’t ram the tag
        Pose2d targetPose = tagPose.plus(
                new Transform2d(
                        new Translation2d(1.5, 0.0), // 1m in front
                        tagPose.getRotation()));

        return AutoBuilder.pathfindToPose(
                targetPose,
                Constants.DriveConstants.PathPlannerConstants.pathConstraints);
    }
}
```

## CAD Files {#cad}

{% if site.features.cad_viewer %}
Phoenix was designed in Onshape with full parametric modeling for easy iteration.

### Design Philosophy
- **Modularity:** Each mechanism can be removed independently
- **Serviceability:** All fasteners accessible without disassembly
- **Weight Distribution:** Low center of gravity for stability

### Key Assemblies

| Assembly | Parts | Weight |
